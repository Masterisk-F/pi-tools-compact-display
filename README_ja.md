# pi-tools-compact-display

Piアシスタント向けの拡張機能です。アシスタントが各種ツール（bash、read、ls、edit など）を実行した際のターミナル（TUI）上での出力表示を、ツールごとに細かくカスタマイズできます。

長すぎるツール出力を制限したり、不要なツール呼び出しを非表示にしてサマリーだけを出力するなど、コンテキストや画面をスッキリと保つのに役立ちます。

## 特徴

- **ツールごとの個別設定**: ツールごとに異なる表示モードを設定可能。
- **サマリー表示**: 実行時は非表示にし、結果だけを「⚡ read(1) ls(2)」のようにまとめて表示。
- **行数制限と空行の省略**: 出力行数を制限したり、上下の不要な空行を自動で削除。
- **外部拡張ツールへの対応**: Pi標準のツールだけでなく、他の拡張機能が追加したツール名も指定して表示を上書き可能。
- **展開への対応**: TUI上で `Ctrl+O` などを押して詳細表示（expanded）を切り替えた際は、制限を解除して全文出力を行います。

## インストールとビルド

```bash
# リポジトリへ移動
cd pi-tools-compact-display

# 依存関係のインストール
npm install

# TypeScriptのビルド
npm run build

# Piコマンドを使用して拡張機能をローカルインストール
pi install .
```

## 設定ファイルの書き方

設定ファイルは以下の場所に `config.json` という名前で作成します。

**ファイルパス:**
`~/.pi/agent/extensions/pi-tools-compact-display/config.json`

### 設定ファイルの構造

設定はJSON形式で、キーに「ツール名」、値にそのツールの「表示設定（オプション）」を記述します。

```json
{
  "ツール名": {
    "mode": "表示モード",
    "outputLines": 行数,
    "noPadding": true または false
  }
}
```

### 設定できる項目

| プロパティ名 | 型 | 説明 |
| --- | --- | --- |
| **`mode`** | `string` | ツールの表示形式。`"count_only"`, `"lines"`, `"default"` のいずれかを指定します。（必須） |
| **`outputLines`** | `number` | 表示する最大行数。`mode` が `"lines"` のときのみ有効です。（任意） |
| **`noPadding`** | `boolean` | 出力の前後に含まれる空行や、連続する空行を削除するかどうか。`true` で空行を省略します。`mode` が `"lines"` のときのみ有効です。（任意） |

### `mode` の種類と挙動

1. **`"count_only"`**
   - **挙動**: ツールが実行されたことや、その出力内容は画面上に一切表示されません。代わりにアシスタントのメッセージの先頭に `⚡ read(1)` のように「実行された回数」がサマリーとしてまとめて表示されます。
   - **用途**: `read`, `ls`, `find`, `grep` など、LLMが自律的に情報を収集するためのツールで、人間がいちいち中身を見る必要がないものに最適です。

2. **`"lines"`**
   - **挙動**: ツールの実行コマンドと、その結果の一部を表示します。`outputLines` で指定した行数のみを表示し、長すぎる出力をカットします。
   - **用途**: `bash`, `edit`, `write` など、LLMがどのような操作を行い、どのようなエラーが出たのかを人間が確認したいツールに最適です。

3. **`"default"`**
   - **挙動**: この拡張機能による表示の書き換えを行わず、Piアシスタントのデフォルトの表示ロジック（または他の拡張機能が定義したロジック）をそのまま適用します。
   - **備考**: 設定ファイルに記述がないツールや、`config.json` 自体が存在しない場合は、全てのツールが自動的に `default` 扱いになります。

---

### 設定例

```json
{
  "read": { "mode": "count_only" },
  "ls": { "mode": "count_only" },
  "find": { "mode": "count_only" },
  "grep": { "mode": "count_only" },
  
  "bash": { 
    "mode": "lines", 
    "outputLines": 3, 
    "noPadding": true 
  },
  "write": { 
    "mode": "lines", 
    "outputLines": 1, 
    "noPadding": true 
  },
  
  "my_custom_tool": {
    "mode": "count_only"
  }
}
```

*※ `my_custom_tool` のように、Pi標準ツール以外の別の拡張機能で追加されたツール名であっても指定可能です。*

## 使い方・ショートカット

設定が完了したら Pi アシスタントを起動してください。

- **件数表示のクリーンナップ**: `count_only` に設定されたツールは、画面上では `⚡ read(1)` のように表示されますが、裏側でLLMに会話履歴（コンテキスト）が送られる際には、このサマリー文字列は自動的に削除されるため、LLMの思考を邪魔しません。
- **全文表示 (Expanded)**: `lines` モードで出力が省略されている場合、Piの標準機能である表示の展開ショートカット（一般的には `Ctrl+O`）を押すことで、省略されていた全文（上限あり）を展開して確認することができます。
